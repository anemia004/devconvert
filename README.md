# DevConvert

A fast, modern web tool for converting between binary, hexadecimal, decimal, and text formats, with built-in Base64 encoding and decoding.

## Features

- Binary ↔ Hex ↔ Decimal ↔ Text conversion
- Base64 encode and decode tool
- Instant conversion with real-time output
- UTF-8 safe handling
- Responsive and clean user interface
- Fully frontend-based (no backend required)

## Built With

- HTML
- CSS
- JavaScript (Vanilla)

## Usage

1. Open `index.html` in a browser  
2. Or deploy using GitHub Pages  
3. Enter input and view instant conversions
